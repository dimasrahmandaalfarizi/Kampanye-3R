# Images Folder

Tempatkan gambar poster kampanye Anda di sini dengan nama `poster.jpg`.

Format yang disarankan:
- Format: JPG atau PNG
- Rasio: 3:4 (portrait) atau sesuai kebutuhan
- Ukuran: Disarankan maksimal 2MB untuk performa optimal

File akan ditampilkan di halaman `/campaign` menggunakan komponen `PosterDisplay`.
